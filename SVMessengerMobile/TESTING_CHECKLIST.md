# Тестов Чеклист - SVMessenger Mobile

## Основни Функционалности

### 1. Infinite Scroll за Съобщения
- [ ] Скролване нагоре в чат зарежда по-стари съобщения
- [ ] Показва се индикатор за зареждане при зареждане на по-стари съобщения
- [ ] Няма дублиране на съобщения
- [ ] Правилно спиране на зареждане когато няма повече съобщения

### 2. Търсене на Разговори
- [ ] Търсене по име на потребител работи
- [ ] Филтриране на разговори в реално време
- [ ] Празно търсене показва всички разговори
- [ ] Правилно показване на резултати

### 3. Панел със Следвани Потребители
- [ ] Показва се списък със следвани потребители в UserSearchScreen
- [ ] Липсва търсене - показва се панелът със следвани
- [ ] Кликване върху следван потребител отваря чат
- [ ] Правилно зареждане и показване на данни

### 4. Детайли за Статус на Съобщение
- [ ] Tap върху статус икона показва модал с детайли
- [ ] Показва се време на изпращане
- [ ] Показва се време на доставка (ако приложимо)
- [ ] Показва се време на прочитане (ако приложимо)
- [ ] Правилно затваряне на модала

### 5. Звуци за Съобщения и Обаждания
- [ ] Звук при ново съобщение (s1.mp3)
- [ ] Звук при входящо обаждане (IncomingCall.mp3)
- [ ] Звук при изходящо обаждане (OutCall.mp3)
- [ ] Звуците спират правилно при отговор/отхвърляне/край на обаждане
- [ ] Звуците не се припокриват

### 6. Reply Функционалност (Message Thread View)
- [ ] Long press върху съобщение показва меню с опция "Отговори"
- [ ] Избор на "Отговори" показва preview на родителското съобщение
- [ ] Preview показва текста на родителското съобщение
- [ ] Може да се отмени reply преди изпращане
- [ ] Изпратеното съобщение показва reply preview
- [ ] Tap върху reply preview скролва до родителското съобщение
- [ ] Правилно изпращане на reply с parentMessageId

### 7. Push Notifications за Обаждания
- [ ] При входящо обаждане когато приложението е затворено се получава notification
- [ ] При отваряне на приложението от notification се показва IncomingCallScreen
- [ ] Звукът за входящо обаждане се възпроизвежда правилно
- [ ] Правилно показване на име на звънящия в notification

### 8. UI/UX Консистентност
- [ ] Всички икони са от Heroicons (без emoji)
- [ ] Консистентни цветове и градиенти
- [ ] Правилни сенки и spacing
- [ ] Консистентен стил на бутони и input полета
- [ ] Правилно показване на reply preview в MessageBubble
- [ ] Правилно показване на reply preview в MessageInput

## Технически Тестове

### Backend Integration
- [ ] API зареждане на съобщения с pagination работи
- [ ] API за изпращане на reply съобщения работи
- [ ] WebSocket получава правилно reply съобщения с parentMessageId и parentMessageText
- [ ] API за следвани потребители работи

### State Management
- [ ] Zustand store правилно управлява pagination state
- [ ] Правилно добавяне на по-стари съобщения без дублиране
- [ ] Правилно управление на reply state в ChatScreen

### Performance
- [ ] Infinite scroll не причинява lag
- [ ] Правилно memoization на компоненти
- [ ] Няма memory leaks при скролване
